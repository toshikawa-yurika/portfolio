コメント

尾本さんや菊池さんが参考サイトにするとおっしゃっていたmanaさんのコードペンから参考にしました。

0% { transform: rotate(0deg); }
100% {transform: rotate(360deg); }で回転させています。

もともとmanaさんのコードペンでは、to {transform: rotate(360deg); }となっており、よくわからなかったのですが、 0% = from , 100% = toでもできるらしいです。 (西本さんに教えてもらいました。)

ちなみに、rotateXにすると前方向に回るので面白かったです。

manaさんのコードペン
  • やりたかった理由
    メイン画像で使おうか検討していたため。
  • 疑問点
    mix-blend-mode: multiplyが調べてもよくわからなかった。
    疑問点2
    このWEBページで実装しようとしたところ、アニメーションでtransformを使ったら、 丸の中央揃えのために使っていたtransform: translate(-50%, -50%);が効かなくなってしまった。
    （なので、このサイトではアニメーション部分はコメントアウトしています。）

    →アニメーション部分でもtransformプロパティを使っていて上書きされてしまっていたのが原因。transform-origin: 0 0;を設定し、アニメーションにもtranslate(-50%, -50%)を設定することで解決した。

  • ポイント
    アニメーションで３つの丸の回転速度をそれぞれ変えている
